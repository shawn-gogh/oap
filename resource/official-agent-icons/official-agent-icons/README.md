| LangGraph | `langgraph/langgraph-logo.svg` | `langchain-ai/agent-chat-ui` | 从官方 React SVG 组件无损导出 |
| CrewAI | `crewai/crewai-icon.svg` | `crewAIInc/crewAI` | 官方文档 favicon 矢量；仓库未提供字标 SVG |
| CrewAI | `crewai/crewai-logo.png`、`crewai/crew-only-logo.png` | `crewAIInc/crewAI` | 官方 PNG 字标，作为矢量图标的补充 |
| A2A | `a2a/*.svg` | `a2aproject/A2A` | 官方彩色、黑、白图标与横版 Logo |

## 使用提示

- UI 小尺寸展示优先使用 `*-icon.svg` / `*-mark.svg`；标题和品牌展示使用 `*-logo.svg`。
- 白色版本用于深色背景，黑色版本用于浅色单色场景。
- ACP 已并入 Linux Foundation 的 A2A 项目；这里保留的是 ACP 历史独立标识。
- Logo 仍受各项目商标政策约束。本包仅重新整理官方原始素材，没有转让商标使用权。
